# Landbot (landbot)

Landbot is a no-code conversational AI platform that enables businesses to build, deploy, and manage chatbots across web, WhatsApp, and messenger channels. The platform provides a REST API for programmatic control of chatbot flows, customer data, webhooks, and channel configuration. Developers can integrate Landbot with external systems using token-based authentication and event-driven webhooks.
